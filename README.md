# Knok Landing

Landing page for [Knok](https://github.com/TomasWard1/knok) — the macOS menu bar app that lets AI agents interrupt you.

## Stack

- **Framework:** Astro
- **CSS:** Tailwind v4
- **Deployment:** Vercel

## Development

```bash
npm install
npm run dev
```
